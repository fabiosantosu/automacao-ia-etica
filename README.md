# Automação de Processos e Engenharia de Prompt Ética

Este repositório reúne projetos focados em transformar fluxos de trabalho manuais em operações inteligentes e responsáveis utilizando LLMs (Large Language Models) e ferramentas low-code.

## 🚀 Projetos Incluídos

### Projeto 1: Triagem Inteligente de Leads
Solução desenvolvida para agências de marketing digital que sofrem com o alto volume de contatos desqualificados. O sistema utiliza IA para analisar a intenção de compra e urgência em tempo real.

* **Impacto:** Redução do tempo de resposta comercial e foco total em leads com alto potencial de fechamento.
* **Stack:** Webhooks, OpenAI API (GPT-4o), Google Sheets.

### Projeto 2: Agente de Recrutamento Ético (EthosTech)
Design de um agente de IA para triagem inicial de currículos em tecnologia, projetado para mitigar vieses algorítmicos e garantir conformidade com a LGPD.

* **Impacto:** Promoção de diversidade no setor tech e segurança jurídica no tratamento de dados sensíveis.
* **Diferencial:** Implementação de Blind Screening e IA Explicável (XAI).
